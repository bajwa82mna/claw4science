About Claw4Science

Connecting the scientific research community with the most powerful AI agent tools in the OpenClaw ecosystem.

Our Mission

Claw4Science exists because the OpenClaw AI agent ecosystem is growing rapidly — too rapidly for most researchers to track. New projects launch weekly, existing ones evolve significantly, and distinguishing genuinely useful scientific tools from hobbyist experiments is time-consuming work. We do that work so you don't have to.

We curate, test, and classify OpenClaw ecosystem projects with a specific focus on scientific research use cases: bioinformatics, biomedicine, drug discovery, autonomous literature review, and AI-assisted experimentation. Our editorial standards prioritize open-source projects with active maintenance, reproducible workflows, and genuine utility for working researchers.

Principles

How We Curate

🔍
Research-First Filtering
We only list projects with clear scientific utility — not every OpenClaw fork makes the cut. Projects must demonstrate real value for researchers.
✅
Verified Open Source
Every project is confirmed open source with an accessible GitHub repository. We link directly to source code, not landing pages.
⚡
Active Maintenance
We track maintenance status (Active/Stale/Dormant) so researchers know what's still being developed versus abandoned projects.
🎯
Honest Assessment
Where projects share names or overlapping scope, we add context. We flag misleading claims and note when "Misleading" status applies.
🌏
Global Coverage
We actively track projects from academic groups worldwide — Tsinghua, Stanford, MIT, Princeton, Alibaba, NVIDIA, and many others.
🔄
Regular Updates
The directory is updated weekly. Star counts refresh periodically. Monthly ecosystem reports summarize major developments.
History

How We Got Here

Late 2023
OpenClaw reaches 100K stars
OpenClaw's rapid growth triggers an explosion of derivative projects — many science-focused teams begin adapting it for research workflows.
Early 2024
ChemCrow launches
ur-whitelab's chemistry AI agent demonstrates that science-specific OpenClaw adaptations can reach production quality and attract serious attention.
Mid 2024
Bioinformatics agent proliferation
AutoBA, ClawBio, CRISPR-GPT, and CellVoyager launch within months of each other. Researchers start asking: where do I find all of these in one place?
Late 2024
Claw4Science launches
We launched the directory with 40 initial projects and a commitment to weekly updates and editorial integrity. claw4science.org goes live.
2025
Rapid ecosystem growth
EvoScientist, AgentLaboratory, STELLA, and Biomni launch. The directory grows to 95+ projects. Monthly ecosystem reports begin.
April 2026
Campus In China mirror launches
claw4science.campusinchina.com goes live — a mirror and research-community edition of the directory optimized for researchers in China.

What Gets Listed

Projects must meet these criteria to be included in the main directory:

  • Open source with public GitHub repository
  • Related to OpenClaw or scientific AI agent use cases
  • Demonstrates meaningful utility for researchers (not toy demos)
  • Has at least some documentation or README explaining usage
  • Is not an exact copy of another listed project without differentiation

Projects that don't meet the bar for the main directory may appear in our Watching List — we monitor them and promote to the main directory when they mature.
